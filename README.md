# 📊 Online Store Executive Dashboard & Customer Analytics
🚀 **Live Demo:** [Click here to view the interactive dashboard](https://retail-customer-analytics-dt9g2qqnbbwtomda5bvmsk.streamlit.app/)

## 📝 Project Overview

This project analyzes **R8.9M in revenue** from a large-scale e-commerce dataset (397,000+ transactions). I built an interactive dashboard to help store managers identify high-value "Whale" customers, track hourly peak shopping trends, and optimize marketing spend.

## 🚀 Key Features

- **Interactive Streamlit Dashboard:** Real-time filtering by shopping hour.
- **RFM Insights:** Identified top-tier customers contributing to 15% of total revenue.
- **Time-Series Analysis:** Discovered the "Golden Hour" for sales (12:00 PM).
- **Top Product Tracking:** Visualized the top 10 revenue-generating items.

## 💡 Business Insights

1. **The Lunchtime Surge:** Peak transaction volume occurs at **12:00 PM**, suggesting that customers shop primarily during work breaks.
2. **Whale Concentration:** A small group of customers (e.g., ID 14646) generates a disproportionate amount of revenue, requiring a specialized loyalty strategy.
3. **Product Pareto:** The "Paper Craft, Little Birdie" is the top revenue driver, significantly outperforming other inventory categories.

## 🛠️ Tech Stack

- **Language:** Python 3.11
- **Libraries:** Pandas (Data Wrangling), Plotly (Interactive Charts), Streamlit (Dashboarding)
